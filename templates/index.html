Vulnerability Categories

SQL Injection

Critical

Direct SQL injection vulnerabilities in user lookup and search functionality.

  • GET /api/user/1 - Direct parameter injection
  • GET /search - Search injection
  • Test: 1 OR 1=1
  • Test: Boolean-based

Authentication & IDOR

Critical

Weak authentication and Insecure Direct Object Reference vulnerabilities.

  • POST /login - Weak validation (admin/admin123)
  • GET /api/profile/1 - IDOR vulnerability
  • GET /api/profile/2 - Access other profiles
  • GET /dashboard/1 - Unauthorized access

Cross-Site Scripting (XSS)

High

Reflected XSS vulnerabilities with no input validation.

  • GET /xss?message=Hello - Parameter reflection
  • Test: Script injection
  • Test: Image tag

Information Disclosure

High

Exposed sensitive files, configs, and debug information.

  • GET /admin - Unprotected admin panel
  • GET /backup - Exposed backup files
  • GET /robots.txt - Sensitive path disclosure
  • GET /sitemap.xml - Site enumeration
  • GET /.git/config - Git directory exposure

Unrestricted HTTP Methods

Medium

Resource endpoints accepting all HTTP methods without validation.

  • GET /resource/123 - Basic GET
  • PUT /resource/123 - Update resource
  • DELETE /resource/123 - Delete resource
  • PATCH /resource/123 - Patch resource

Unvalidated Redirects

Medium

Unvalidated redirect parameters leading to external sites.

  • GET /redirect - Redirect to external site
  • Test: Phishing
